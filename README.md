# دفتر مصروفاتي (Daftar Masrofaty)

تطبيق Flutter/Dart بسيط لتتبع المرتب والمصاريف الشهرية، بنفس فكرة ملف الإكسل: تسجيل مصروف يومي، فئات، وملخص شهري بقاعدة 50/30/20.

## المتطلبات قبل البناء

على جهازك (وليس هنا)، لازم يكون عندك:

1. **Flutter SDK** مثبت: https://docs.flutter.dev/get-started/install
2. **Android SDK** (بييجي تلقائي مع Android Studio): https://developer.android.com/studio
3. تأكد إن الإعداد شغال:
   ```
   flutter doctor
   ```
   لازم تشوف علامة ✓ جنب Android toolchain.

## خطوات التشغيل

1. فك ضغط المشروع، وادخل المجلد:
   ```
   cd flutter_app
   ```
2. نزّل المكتبات:
   ```
   flutter pub get
   ```
3. لتجربة التطبيق على موبايل متوصل بالكابل أو محاكي:
   ```
   flutter run
   ```
4. لبناء ملف APK جاهز للتثبيت:
   ```
   flutter build apk --release
   ```
   الملف هيطلع في:
   ```
   build/app/outputs/flutter-apk/app-release.apk
   ```
   انقله لموبايلك وثبّته (لازم تسمح بـ "تثبيت من مصادر غير معروفة" في إعدادات الأندرويد).

## بنية المشروع

```
lib/
  main.dart                 نقطة الدخول + الألوان والثيم
  models/expense.dart       نموذج بيانات المصروف والفئات
  services/storage_service.dart   الحفظ المحلي (shared_preferences)
  screens/home_screen.dart  الشاشة الرئيسية (تابات: الملخص / السجل / الإعدادات)
```

## بناء الـ APK أونلاين (من غير Flutter SDK على جهازك)

المشروع فيه ملفين إعداد جاهزين لكده: `codemagic.yaml` و `.github/workflows/build.yml`.

### الطريقة 1: Codemagic (الأسهل)

1. ارفع مجلد المشروع كـ repo على GitHub (لازم يكون فيه `pubspec.yaml` في الجذر).
2. سجّل دخول على codemagic.io بحساب GitHub.
3. من "Add application" اختار الـ repo بتاعك.
4. Codemagic هيكتشف إنه مشروع Flutter تلقائي ويستخدم `codemagic.yaml` الموجود.
5. دوس "Start new build" واختار workflow اسمه `android-build`.
6. بعد ما البيلد يخلص (٥-١٠ دقايق)، هتلاقي ملف الـ APK في تبويب Artifacts تقدر تنزّله على طول.
7. فيه باقة مجانية (500 دقيقة بناء شهرياً وقت كتابة الكلام ده) كفاية لمشروع بسيط زي ده.

### الطريقة 2: GitHub Actions (مجاني بالكامل)

1. ارفع المشروع كـ repo على GitHub.
2. الملف `.github/workflows/build.yml` هيشتغل تلقائي مع أول push.
3. من تبويب "Actions" في الـ repo، افتح آخر تشغيل، وانزل لقسم "Artifacts" في الآخر، هتلاقي `app-release-apk` تقدر تنزّله كملف zip فيه الـ APK.

كلا الطريقتين مجانيتين لمشروع بالحجم ده ومش محتاجين منك أي إعداد إضافي غير رفع الكود على GitHub.

- التطبيق بالكامل RTL وباللغة العربية.
- لو حبيت تغيّر اسم التطبيق أو الأيقونة قبل البناء، عدّل `android/app/src/main/AndroidManifest.xml` (اسم التطبيق) وملفات الأيقونات في `android/app/src/main/res/mipmap-*`.
