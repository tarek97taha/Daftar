# دفتر مصروفاتي (Daftar Masrofaty)

تطبيق Flutter/Dart بسيط لتتبع المرتب والمصاريف الشهرية، بنفس فكرة ملف الإكسل: تسجيل مصروف يومي، فئات، وملخص شهري بقاعدة 50/30/20.

## المتطلبات قبل البناء

على جهازك (وليس هنا)، لازم يكون عندك:

1. **Flutter SDK** مثبت: https://docs.flutter.dev/get-started/install
2. **Android SDK** (بييجي تلقائي مع Android Studio): https://developer.android.com/studio
3. تأكد إن الإعداد شغال:
   ```
   flutter doctor
   ```
   لازم تشوف علامة ✓ جنب Android toolchain.

## خطوات التشغيل

1. فك ضغط المشروع، وادخل المجلد:
   ```
   cd flutter_app
   ```
2. نزّل المكتبات:
   ```
   flutter pub get
   ```
3. لتجربة التطبيق على موبايل متوصل بالكابل أو محاكي:
   ```
   flutter run
   ```
4. لبناء ملف APK جاهز للتثبيت:
   ```
   flutter build apk --release
   ```
   الملف هيطلع في:
   ```
   build/app/outputs/flutter-apk/app-release.apk
   ```
   انقله لموبايلك وثبّته (لازم تسمح بـ "تثبيت من مصادر غير معروفة" في إعدادات الأندرويد).

## بنية المشروع

```
lib/
  main.dart                 نقطة الدخول + الألوان والثيم
  models/expense.dart       نموذج بيانات المصروف والفئات
  services/storage_service.dart   الحفظ المحلي (shared_preferences)
  screens/home_screen.dart  الشاشة الرئيسية (تابات: الملخص / السجل / الإعدادات)
```

## المميزات

- تسجيل مصروف: البند، الفئة، المبلغ، التاريخ
- 11 فئة جاهزة (أكل، مواصلات، سكن، فواتير، اشتراكات، تسلية، تسوق، خروجات، ادخار، سداد ديون، تانية)
- ملخص شهري: إجمالي المصروفات، المتبقي من المرتب
- مقارنة تلقائية بقاعدة 50/30/20 (احتياجات / رغبات / ادخار) مع أشرطة تقدّم
- كل البيانات محفوظة محلياً على الجهاز (مفيش سيرفر أو إنترنت مطلوب)

## ملاحظات

- التطبيق بالكامل RTL وباللغة العربية.
- لو حبيت تغيّر اسم التطبيق أو الأيقونة قبل البناء، عدّل `android/app/src/main/AndroidManifest.xml` (اسم التطبيق) وملفات الأيقونات في `android/app/src/main/res/mipmap-*`.
- الملف ده كود مصدري فقط — لسه محتاج تعمل `flutter build apk` بنفسك عشان يطلع ملف APK فعلي، لأن ده محتاج Flutter SDK و Android SDK على جهازك.
